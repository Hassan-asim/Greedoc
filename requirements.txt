## Installation Instructions
1. Clone the repository
2. Install backend dependencies: cd server && npm install
3. Install frontend dependencies: cd client && npm install
4. Copy setup/env.example to .env and fill in your values
5. Place your Firebase service account JSON file in the root directory
6. Start the backend: cd server && npm start
7. Start the frontend: cd client && npm run dev
